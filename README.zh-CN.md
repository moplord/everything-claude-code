# Codex 优先仓库

这个仓库以 **OpenAI Codex** 为主进行维护。

入口：
- 项目指令：`AGENTS.md`
- 项目技能：`.codex/skills/`

## 仓库内容

- Codex-first 的「需求 -> JDL ->（JHipster）生成代码」工作流，以及可复用的 module packs。
- 从上游同步下来的 Claude Code 相关材料被保存在 `legacy/claude-code/` 仅作参考，**Codex 不使用**。

## 快速开始（Codex）

1) 用 Codex 打开本仓库。
2) 按 `AGENTS.md` 的约定执行。
3) 使用 `.codex/skills/` 下的 skills（例如 `$requirements-authoring`、`$jdl-derivation`、`$jhipster-codegen`、`$module-pack`）。
